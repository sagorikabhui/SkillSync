# SkillSync

SkillSync is a student collaboration platform built using Django.

## Features

* Student Profiles
* Skill Discovery
* Project Collaboration
* Study Groups
* Dark Mode
* Search Students
* Authentication System

## Tech Stack

* Django
* Python
* HTML
* CSS
* JavaScript
* SQLite

## Future Enhancements

* Real-time Group Chat
* Resource Sharing
* Group Announcements
* Mentor System
